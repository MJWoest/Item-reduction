These .R files describe how the results of the research article "Item reduction of the Patient Reported Wrist Evaluation using decision tree modelling" can be reproduced.

The 'PRWHE pijn in R' file can be used as a model for other questionnaires, 
the 'PRWHE function in R' file is provided to complete all code used in the article, but is nearly the same.

The output of the 'PRWHE pijn in R' file is versatile; It wil produce a decision tree, but is also able to optimize parameters used to create the dicision tree.
